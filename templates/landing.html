// how it works

Simple. Fast. Reliable.

From URL list to PDF report in under 30 seconds.

01

Paste your URLs

Copy and paste up to 50 URLs into the form — one per line. Give your report a name or leave it blank.

02

We check everything

Our crawler visits each URL, captures the HTTP status, page title, meta description and response time.

03

Review and filter

See all results in a clean table. Filter by status — show only errors to find broken links instantly.

04

Download your PDF

Export a formatted PDF report ready to share with your client or team. One click, done.

import requests

# Authenticate
token = requests.post("/api/v1/auth/login/", json={
    "username": "you",
    "password": "••••••••"
}).json()["access"]

# Create a report
report = requests.post("/api/v1/reports/create/",
    headers={"Authorization": f"Bearer {token}"},
    json={"urls": "https://site.com\nhttps://site.com/about"}
).json()

print(report["id"])  # ready to poll
